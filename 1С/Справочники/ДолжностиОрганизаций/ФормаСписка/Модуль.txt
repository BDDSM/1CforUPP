////////////////////////////////////////////////////////////////////////////////
// ПРОЦЕДУРЫ И ФУНКЦИИ ОБЩЕГО НАЗНАЧЕНИЯ

Процедура УстановитьЗаголовкиОрганизацийВФорме()
	
	ЭтаФорма.Заголовок = ОбщегоНазначения.ПреобразоватьСтрокуИнтерфейса("Должности организаций");
	
КонецПроцедуры // УстановитьЗаголовкиОрганизацийВФорме()

////////////////////////////////////////////////////////////////////////////////
// ПРОЦЕДУРЫ - ДЕЙСТВИЯ КОМАНДНЫХ ПАНЕЛЕЙ ФОРМЫ

// Обработчик события Действие элемента КоменднаяПанель.ДействиеПодбор.
//
Процедура КоманднаяПанельФормаДействиеПодбор(Кнопка)

	ИмяМакета = Кнопка.Имя;
	Справочники.ДолжностиОрганизаций.ПолучитьФорму("ФормаПодбораИзКлассификатора", ЭтаФорма, "ФормаПодбораИзКлассификатора").Открыть();

КонецПроцедуры

Процедура ДополнительныеДействия(Кнопка)
	
	ДолжностиОрганизацийПереопределяемый.ВыполнитьДополнительныеДействия(ЭтаФорма, Кнопка.Имя);
	
	ИзменениеАктуальностиЭлементовПереопределяемый.ВыполнитьДополнительныеДействияФормыСписка(Кнопка, ЭтаФорма, СправочникСписок);
	
КонецПроцедуры

////////////////////////////////////////////////////////////////////////////////
// ПРОЦЕДУРЫ - ОБРАБОТЧИКИ СОБЫТИЙ ФОРМЫ

Процедура ПередОткрытием(Отказ, СтандартнаяОбработка)
	
	ДолжностиОрганизацийПереопределяемый.ФормаСпискаПередОткрытиемДополнительно(ЭтаФорма, Новый Действие("ДополнительныеДействия"));
	
	ИзменениеАктуальностиЭлементовПереопределяемый.ФормаСпискаПередОткрытиемДополнительно(ЭтаФорма, Новый Действие("ДополнительныеДействия"), СправочникСписок);
	
КонецПроцедуры

Процедура ПриОткрытии()
	
	ДолжностиОрганизацийПереопределяемый.ПриОткрытииДополнительно(ЭтаФорма);
	
	УстановитьЗаголовкиОрганизацийВФорме();
	
	ИзменениеАктуальностиЭлементовПереопределяемый.ФормаСпискаПриОткрытииДополнительно(ЭтаФорма, СправочникСписок);
	
КонецПроцедуры

Процедура ОбработкаВыбора(ЗначениеВыбора, Источник)
	
	ДолжностиОрганизацийПереопределяемый.ФормаСпискаОбработкаВыбораДополнительно(ЭтаФорма, ЗначениеВыбора, Источник);
	
КонецПроцедуры

Процедура ПриЗакрытии()
	
	ИзменениеАктуальностиЭлементовПереопределяемый.ФормаСпискаПриЗакрытииДополнительно(ЭтаФорма, СправочникСписок);
	
КонецПроцедуры

////////////////////////////////////////////////////////////////////////////////
// ПРОЦЕДУРЫ - ОБРАБОТЧИКИ СОБЫТИЙ ЭЛЕМЕНТОВ ФОРМЫ

Процедура СправочникСписокПриПолученииДанных(Элемент, ОформленияСтрок)
		
	ИзменениеАктуальностиЭлементовПереопределяемый.ФормаСпискаОформлениеСпискаДополнительно(ЭтаФорма, Элемент, ОформленияСтрок);

КонецПроцедуры
